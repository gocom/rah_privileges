h1. rah_privileges

"Download":https://github.com/gocom/rah_privileges/releases | "Packagist":https://packagist.org/packages/rah/rah_privileges | "Donate":http://rahforum.biz/donate/rah_privileges

Configure admin-side user-group permissions from Textpattern's Advanced Preferences panel.

h2. Install

Using "Composer":http://getcomposer.org:

bc. $ composer require rah/rah_privileges:*

Or "download":https://github.com/gocom/rah_privileges/releases a plugin package.

h2. Requirements

* Textpattern 4.4.1 or newer.
* PHP 5.2.0 or newer.

h2. Toolshed notice

This is a toolshed project. Experimental and not part of the main supported product line of Rah.
